# 📐 Landing Page — Curso de Matemática PM-SP

Landing page desenvolvida para o **Prof. Thiago Henrique**, especialista em matemática para concursos públicos, com foco no concurso de Soldado da PM-SP.

Projeto desenvolvido como parte do meu portfólio de desenvolvimento front-end.

---

## 🚀 Tecnologias utilizadas

- **HTML5** — estrutura semântica da página
- **CSS3** — estilização, animações e responsividade
- **JavaScript** — carregamento dinâmico de conteúdo (grade curricular e FAQ)

---

## ✨ Funcionalidades

- Layout totalmente responsivo (mobile, tablet e desktop)
- Grade curricular carregada dinamicamente via JavaScript
- Seção de FAQ com accordion (abrir/fechar respostas)
- Animações em botões com efeito hover

## 📱 Responsividade

A página foi desenvolvida com abordagem **desktop-first**, utilizando media queries para adaptar o layout a telas menores:

| Breakpoint | Layout |
|---|---|
| `> 768px` | Desktop — duas colunas, grid de 3 colunas |
| `≤ 768px` | Mobile — coluna única, elementos empilhados |
